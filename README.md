# Django React

This is a very lean implementation of React and python Django. 

##Requirements

* Node
* Npm
* Python 

##Installation

* `cd djangoreact`
* `source gjr_env/bin/activate`
* `git rm .git && git init`
* `npm install`
* `npm run webpack`
* `pip install -r requirements.txt`
* `python manage.py runserver`
